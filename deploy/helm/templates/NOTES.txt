Release:   {{ .Release.Name }}
Namespace: {{ .Release.Namespace }}
Chart:     {{ .Chart.Name }}-{{ .Chart.Version }} (app {{ .Chart.AppVersion }})

1. Verify the pod is ready:

   kubectl --namespace {{ .Release.Namespace }} rollout status deploy/{{ include "nimiq-simple-faucet.fullname" . }}

2. Port-forward the service for local access:

   kubectl --namespace {{ .Release.Namespace }} port-forward svc/{{ include "nimiq-simple-faucet.fullname" . }} 8080:{{ .Values.service.port }}

   Then open http://localhost:8080/

3. Fetch the initial admin password (only when the secret is managed in-cluster):

   kubectl --namespace {{ .Release.Namespace }} get secret {{ include "nimiq-simple-faucet.secretName" . }} \
     -o jsonpath='{.data.FAUCET_ADMIN_PASSWORD}' | base64 -d ; echo

4. Sign in to the admin UI and complete TOTP setup:

   http://<host>/admin

5. Fund the faucet hot wallet shown in the dashboard. Until it is funded,
   `/v1/claim` will return HTTP 503.
