# Examples

Runnable example projects demonstrating faucet integration across frameworks.

Each example includes Docker support and can be started with docker compose.

| Example | Framework | Description |
|---------|-----------|-------------|
| [Next.js](./nextjs) | React | Claim page with `useFaucetClaim` hook |
| [Vue](./vue) | Vue 3 | Claim page with composables |
| [Capacitor](./capacitor) | Capacitor | Hybrid mobile with device fingerprint |
| [Flutter](./flutter) | Flutter | Dart mobile app |
| [Go Backend](./go) | Go | Server-side proxy pattern |
| [Python Backend](./python) | Python | Backend integration |
