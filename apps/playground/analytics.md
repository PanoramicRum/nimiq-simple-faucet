# Analytics

Live faucet analytics are available at the ClaimUI status page.

## What's tracked

- **Current balance** — real-time faucet wallet balance
- **Claim counters** — successful claims over 24h / 7d / 30d
- **Abuse prevention** — blocked attempts over 24h / 7d / 30d
- **Recent claims** — live feed of the latest transactions
- **System health** — node sync status, driver readiness

## Access

The analytics dashboard is part of the [ClaimUI](/paths/quick-demo) application and is available at the `/status` route of any deployed faucet instance.
